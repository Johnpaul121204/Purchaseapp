# Purchase-app

React E-Commerce Frontend is a simple shopping web application built with React.js and Vite. It allows users to browse products, view product details, and manage items in a cart. The project demonstrates component-based architecture, React Router navigation, and Context API for global cart state management.

## Features
- Product listing
- Product details page
- Add to cart functionality
- Remove items from cart
- Cart total calculation

## Technologies Used
- React.js
- Vite
- JavaScript
- React Router
- Context API
- CSS
